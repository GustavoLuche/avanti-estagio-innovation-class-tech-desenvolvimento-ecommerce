## Objetivo deste repositorio

Concentrar os dois projetos solicitados no processo seletivo em um unico repositorio publico, com estrutura clara, instrucoes de execucao e evidencias de funcionamento.

## Estrutura esperada

```text
.
|- projeto-01-desenvolvimento-layout/
|- projeto-02-busca-perfil-github/
|- proxima-etapa-testes-tecnicos.md
|- README.md
```

## O que o repositorio de entrega deve conter

### Obrigatorio

- Projeto 01 completo (HTML, CSS, JavaScript)
- Projeto 02 completo (React + consumo da API do GitHub)
- Fidelidade ao layout do Figma em desktop e mobile
- Tratamento de erro na busca de perfil (Projeto 02)
- Repositorio publico no GitHub

### Recomendado para avaliacao

- README em cada projeto com:
	- Tecnologias usadas
	- Como instalar dependencias
	- Como executar localmente
	- Como gerar build (quando aplicavel)
- Capturas de tela ou GIF curto demonstrando funcionamento
- Codigo organizado e com nomes de arquivos/pastas claros

## Como o avaliador deve encontrar cada projeto

- Projeto 01: pasta `projeto-01-desenvolvimento-layout`
- Projeto 02: pasta `projeto-02-busca-perfil-github`

Cada pasta deve ser executavel de forma independente.
