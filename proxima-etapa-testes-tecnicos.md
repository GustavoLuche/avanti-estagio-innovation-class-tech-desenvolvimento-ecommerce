## Projeto 02 - Busca de Perfil no GitHub (React)

### Objetivo

Criar aplicacao React para buscar usuario no GitHub e exibir dados conforme Figma.

Figma do Projeto 02:
https://www.figma.com/design/ct69qBAokxqyAYBQT5RRcq/innovation-class--Copy---Copy-?node-id=22-2864&p=f&t=jpMRKonoJfklZiNH-0

### Requisitos obrigatorios

- Campo de busca por username do GitHub
- Consumo da API do GitHub no clique de busca
- Exibicao de:
  - Nome
  - Foto de perfil

- Mensagem de erro para perfil nao encontrado
- Layout seguindo o Figma
- Framework CSS opcional (Tailwind, Bootstrap ou outro)

### Opcional

- Loading durante requisicao
- Efeitos visuais para melhorar a experiencia
